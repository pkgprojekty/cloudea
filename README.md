# Strategy & Architecture — Premium Website

## Fixed deployment notes
This package includes:
- `'use client'` added to `app/page.js`
- Next.js upgraded to `14.2.35` to address the December 11, 2025 security advisory

## Local run
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Upload this folder to GitHub or drag-and-drop it into Vercel.
2. Vercel will detect Next.js automatically.
3. Click **Deploy**.
